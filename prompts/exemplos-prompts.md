# 🤖 Engenharia de Prompts

## Prompt 1 - Conceito Básico

### Pergunta

Explique IA Generativa para uma pessoa sem conhecimento técnico.

### Resposta Obtida

A IA Generativa é uma tecnologia capaz de criar novos conteúdos como textos, imagens e códigos a partir de padrões aprendidos durante o treinamento.

### Avaliação

Resposta clara e objetiva.

## Prompt 2 - Comparação

### Pergunta

Compare IA Tradicional e IA Generativa em formato de tabela.

### Resultado

| IA Tradicional | IA Generativa |
|---------------|---------------|
| Classifica dados | Cria conteúdos |
| Predição | Geração |
| Automação | Criatividade assistida |

### Avaliação

Excelente visualização das diferenças.
